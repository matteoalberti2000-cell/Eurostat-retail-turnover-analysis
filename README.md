# Retail Turnover Analysis in Europe

## Overview

This project analyzes retail turnover trends in Italy, France and Germany using Eurostat data.

## Objectives

- Clean and transform Eurostat data
- Analyze long-term retail turnover trends
- Measure the impact of major economic crises
- Compare economic resilience across countries

## Tools

- Python
- Pandas
- NumPy
- Matplotlib

## Dataset

Source: Eurostat

## Main Findings

- Italy was strongly affected by the 2008 Financial Crisis, the European Sovereign Debt Crisis and Covid-19.
- Germany displayed the fastest recovery times.
- Recovery time provides a useful proxy for economic resilience.
